# pkView
Android  双选项pk 投票的 自定义view
Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
Step 2. Add the dependency

	dependencies {
	        implementation 'com.github.nanleiting:pkView:v1.0.0'
	}
